# Arithmetic plugins

```polygolf
$x:-oo..oo <- 0;
~ $x;
```

```polygolf arithmetic.removeBitnot
$x:-oo..oo <- 0;
-1 + (-1 * $x);
```

```polygolf
$x:-oo..oo <- 0;
$x + 1;
```

```polygolf arithmetic.addBitnot
$x:-oo..oo <- 0;
-1 * (~ $x);
```

```polygolf
$x:-oo..oo <- 0;
$x - 1;
```

```polygolf arithmetic.addBitnot
$x:-oo..oo <- 0;
~ (-1 * $x);
```
